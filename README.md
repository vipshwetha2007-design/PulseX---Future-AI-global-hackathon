# ❤️ PulseX AI

> **AI-Gated Emergency Healthcare Platform** built with **React, Vite, Node.js, and Express** for the **Future AI Global Hackathon**.

---

## 📖 Overview

PulseX AI is an AI-powered emergency healthcare platform designed to provide secure, role-based access for **Patients**, **Doctors**, **Ambulance Services**, and **Administrators**. The platform combines a modern React frontend with a robust Express backend to deliver an intuitive and responsive healthcare experience.

---

## ✨ Features

- 🤖 AI-powered Healthcare Assistant
- 🔐 Secure User Authentication
- 👨‍⚕️ Doctor Portal
- 👤 Patient Portal
- 🚑 Ambulance Portal
- 🛡️ Admin Dashboard
- 📱 Responsive User Interface
- ⚡ Fast React + Vite Frontend
- 🚀 RESTful Node.js Backend

# 🚀 Getting Started

## Prerequisites

- Node.js (v18 or later)
- npm

Verify installation:

```bash
node -v
npm -v
```

---

## Clone the Repository

```bash
git clone https://github.com/vipshwetha2007-design/PulseX---Future-AI-global-hackathon.git
cd PulseX---Future-AI-global-hackathon
cd pulsex-improved
```

---

## Install Dependencies

### Backend

```bash
cd backend
npm install
```

### Frontend

Open another terminal:

```bash
cd frontend
npm install
```

---

## ▶️ Run the Application

### Terminal 1 – Backend

```bash
cd backend
npm run dev
```

Backend runs at:

```text
http://localhost:4000
```

---

### Terminal 2 – Frontend

```bash
cd frontend
npm run dev
```

Frontend runs at:

```text
http://localhost:5173
```

Open your browser and visit:

```text
http://localhost:5173
```
