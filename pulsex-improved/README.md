# ❤️ PulseX AI

An AI-powered healthcare platform built using **React**, **Vite**, **Node.js**, and **Express**. PulseX AI provides role-based access for Patients, Doctors, Ambulance services, and Admins through a modern and responsive web application.

---

## ✨ Features

- 🔐 Secure Authentication
- 👨‍⚕️ Doctor Portal
- 🧑‍🤝‍🧑 Patient Portal
- 🚑 Ambulance Portal
- 🛡️ Admin Dashboard
- 🤖 AI-powered Healthcare Assistance
- 📱 Responsive User Interface

---

## 🛠️ Tech Stack

### Frontend
- React
- Vite
- JavaScript
- CSS

### Backend
- Node.js

# 🚀 Getting Started

## Prerequisites

Install:

- Node.js (v18 or later)
- npm

Verify installation:

```bash
node -v
npm -v
```

---

## Clone the Repository

```bash
git clone https://github.com/<your-github-username>/<repository-name>.git
cd pulsex-improved
```

Replace `<your-github-username>` and `<repository-name>` with your own after creating the GitHub repository.

---

## Install Dependencies

### Backend

```bash
cd backend
npm install
```

### Frontend

Open a new terminal and run:

```bash
cd frontend
npm install
```

---

## ▶️ Run the Application

### Terminal 1 – Backend

```bash
cd backend
npm run dev
```

Backend runs at:

```
http://localhost:4000
```

---

### Terminal 2 – Frontend

```bash
cd frontend
npm run dev
```

Frontend runs at:

```
http://localhost:5173
```

Open your browser and visit:

```
http://localhost:5173
```
